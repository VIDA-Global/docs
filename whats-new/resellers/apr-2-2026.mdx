# A New Reseller Workspace for Setup, Branding, and Customer Management

We’ve introduced a new reseller experience in VIDA designed to make setup easier, day-to-day administration clearer, and customer management more efficient.

If you manage customers as a reseller, you’ll now see a more complete workspace built specifically for reseller operations, from onboarding and branding to account setup and navigation.

## What’s new

### A dedicated **For Resellers** workspace

Reseller accounts now have a dedicated **For Resellers** area in VIDA. This workspace brings key setup and management tools into one place, including:

- Overview

- Payouts

- Plans

- Domain & Email

- Links & Labels

- Default Settings

- Feature Flags

- Telemetry

- Embed Your App

This gives reseller teams a central hub for configuring and managing their reseller environment.

### Guided reseller onboarding in-product

New reseller users are now guided through setup directly inside VIDA. The onboarding flow walks users through:

- who they build for

- how they work with clients

- client size

- agent categories

- plan selection

Users can also track progress and follow recommended next steps, making it easier to get fully set up.

### Clearer reseller mode and access

Reseller mode is now more explicit across the app. Eligible admins can request or enable reselling, and reseller-only workflows are clearly gated until reseller mode is turned on.

This makes it easier to understand which features are available and what needs to be completed before using reseller-specific tools.

### Smarter organization creation

Creating organizations under a reseller account is now more structured. Resellers can create organizations as either:

- **Internal**

- **Customer**

For customer organizations, reseller mode must be enabled and a plan must be selected. This makes the billing model clearer from the start.

### Faster white-label app launch

The reseller overview now makes it easier to launch your branded app. Reseller users can:

- preview their white-label app

- open it directly

- copy the app URL

- jump straight to domain setup

This helps teams move more quickly from setup to launch.

### Reorganized reseller admin experience

Reseller administration has been reorganized into clearer sections, making it easier to find the tools you need. Admin areas now include:

- General

- Members

- Billing

- Usage

- Security

- API Keys

- Webhooks

- SIP Settings

The result is a cleaner, more intuitive admin experience.

### Better billing and access guardrails

Billing and upgrade flows now better reflect reseller-led account hierarchies. When action is needed, VIDA routes blockers and upgrade prompts to the correct account owner instead of sending users through a generic path.

### A reseller-specific What’s New feed

Reseller users now have their own scoped **What’s New** feed and viewed state, separate from other account types. That means the updates you see are more relevant to your role and workspace.

### Better first-run guidance across key pages

Reseller-and-above users now get better first-run guidance on important pages, including:

- Organizations

- Agents

- Templates

- Logs

- Contacts

- Tasks

This makes it easier to understand what each area is for and what to do next.

### Easier navigation across accounts

We’ve improved navigation for users who work across multiple accounts. Updates include:

- recently viewed accounts

- improved account switching

- **View As** role preview

These changes make it easier for resellers and parent admins to move between customer accounts with more confidence.

## Why this matters

This release gives reseller users a more complete workspace inside VIDA, with guided setup, stronger white-label controls, clearer administration tools, and better navigation for managing multiple accounts.
